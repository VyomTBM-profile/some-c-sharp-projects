# some-c-sharp-projects
Some C# Projects that I made to learn to the programming language.

## Clone Repos
If you want use the C# Projects yourself to make copies or just to **test and compile them.**

### Clone via Git

Paste this command into your native git client:

```git
git clone https://github.com/VyomTBM-profile/some-c-sharp-projects.git
```

### Clone via Github CLI

Paste this command in your **gh CLI**

```bash
gh repo clone VyomTBM-profile/some-c-sharp-projects
```

or you can use **Fork option** in github.
